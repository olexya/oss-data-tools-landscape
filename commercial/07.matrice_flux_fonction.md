# Flow × Function Matrix - Commercial Solutions

> **Objective:** Two-dimensional classification of commercial solutions: Flow Typology AND Function in the data architecture.

## Introduction

This matrix allows the selection of commercial tools according to:
1. **The temporal axis**: Streaming, Micro-Batching, Batching
2. **The functional axis**: Collection, Transport, Storage, Processing, Analysis, Governance
